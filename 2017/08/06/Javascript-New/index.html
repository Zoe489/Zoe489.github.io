new操作符具体干了什么呢？

(1)新建一个空对象o；
(2)设置原型链，将o的__proto__指向Person.prototype,完成对Person的原型对象的属性和方法的继承；
(3)让Person中的this指向o；
(4)返回对象o。
此时调用person()函数就相当于new Person()。
